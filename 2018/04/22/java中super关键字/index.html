java中super关键字

2018-04-22 Java PV:

super 是指当前类或对象的直接父类或父类对象的引用

#1.调用父类的构造方法。

子类中的构造方法总是先显式或隐式的调用父类的构造方法从而初始化子类中继承的父类成员。

#2.访问被隐藏的成员变量和被重写的方法。
